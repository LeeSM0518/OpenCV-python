# Drawing shapes

## OpenCV 그리기 함수들

**cv2.line()** : 직선 그리기 함수

**cv2.circle()** : 원 그리기 함수

**cv2.rectangle()** : 직사각형 그리기 함수

**cv2.putText()** : 텍스트 입력 함수



## 예제

* **code**

  ```python
  import numpy as np
  import cv2
  
  def drawing():
      # 각종 도형을 그리기 위한 공간 생성
      # 이 함수는 numpy 배열을 만들고 모든 값을 0으로 채우는 함수
      # 512 x 512 이고, (0, 0, 0) 인 배열
      # 채우는 데이터 타입은 uint8
      # (0, 0, 0) : 즉 BGR의 값으로 검정색을 뜻한다.
      img = np.zeros((512, 512, 3), np.uint8)
  
      # 직선을 그린다.
      # img : 직선을 그릴 그림한
      # (0, 0) : 직선의 시작점
      # (511, 511) : 직선의 끝점
      # (255, 0, 0) : BGR 값으로 선의 색상
      # 5 : 선의 굵기
      cv2.line(img, (0, 0), (511, 511), (255, 0, 0), 5)
  
      # 사각형 그리기
      # (384, 0) : 좌측 상단 꼭지점
      # (510, 128) : 우측 하단 꼭지점
      # (0, 255, 0) : 색상
      # 3 : 두께
      cv2.rectangle(img, (384, 0), (510, 128), (0, 255, 0), 3)
  
      # 원 그리기
      # (447, 63): 원의 중심
      # 63 : 원의 반지름
      # (0, 0, 255) : 색상
      # -1 : 주어진 색상으로 도형을 채움
      cv2.circle(img, (447, 63), 63, (0, 0, 255), -1)
  
      # 타원 그리기
      # (256, 256) : 타원의 중심
      # (100, 50) : 각각 장축과 단축의 1/2 길이
      # 0, 0, 180 : 타원의 기울기 각도, 그리기 시작 각도, 끝 각도
      cv2.ellipse(img, (256, 256), (100, 50), 0, 0, 180, (255, 0, 0), -1)
  
      # 폰트 지정
      font = cv2.FONT_HERSHEY_SIMPLEX
      # OpenCV : 글자
      # (10, 500) : 위치
      # 4 : 폰트 크기
      # (255, 255, 255) : 글자 색깔
      # 2 : 굵기
      cv2.putText(img, 'OpenCV', (10, 500), font, 4, (255, 255, 255), 2)
  
      cv2.imshow('drawing', img)
      cv2.waitKey(0)
      cv2.destroyAllWindows()
  
  drawing()
  ```
